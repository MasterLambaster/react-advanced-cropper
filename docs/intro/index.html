Getting started

Package installation#

To use the library you should install it with npm or yarn

npm install -S react-advanced-cropper
yarn add react-advanced-cropper

Then import the Cropper component and the styles somewhere:

import { Cropper } from 'react-advanced-cropper'import 'react-advanced-cropper/dist/style.css';

The following example shows the usage of the cropper in a custom component:
